# AI Document Verification System

AI-assisted OCR web application for document verification in low-resource rural communities (Nepal, India, Africa).

## Tech Stack
- Python
- OpenCV
- Tesseract OCR
- Flask

## Features
- Upload document images
- Preprocess low-quality images
- Extract text using OCR
- Confidence scoring
- Human approval workflow

Final-year dissertation project (in development).
